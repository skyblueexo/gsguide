# Double Prompt
* 什麼是Double Prompt？
   * 當玩家被亮出兩張已知同顏色牌時，另個玩家被提示一張同顏色匹配兩張已知同顏色，叫做Double Promt

# Double Finess
* 什麼是Double Finess?
   * 當玩家被亮出一張時另兩個玩家各擁有一張匹配的未知牌
   * 或另個玩家擁有兩張匹配的未知牌
   * 匹配的未知牌必須打出，否則亮出的牌必須是一樣的牌
